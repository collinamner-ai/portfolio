Editing this portfolio

Keep this site image-led. Replace the PNG files in assets/projects/, then update js/content.js if you change titles, categories or file paths.

  1. Put your PNGs into folders like assets/projects/project-01/01.png.
  2. Use four strong images per project where possible.
  3. Keep descriptions to one short sentence.
  4. Do not add long CV copy to the homepage.
